Description : 
Poème généré à partir d'une image. Les paramètres pris en compte sont la luminosité moyenne, la saturation moyenne et les 2 couleurs dominantes de l'image.

Indication :
Cliquer pour regénèrer un nouveau poème.
Modifier le chemin dans preload() pour changer l'image.
